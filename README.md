# Simple Calculator


The Simple Calculator is a basic web calculator built with HTML, CSS, and JavaScript. It allows users to perform simple arithmetic operations, such as addition, subtraction, multiplication, and division. The calculator features a user-friendly interface and is designed for ease of use.

## Features

- **Basic Arithmetic Operations**: Perform addition, subtraction, multiplication, and division.

- **User-Friendly Interface**: A simple and intuitive user interface for entering and calculating numbers.

- **Clear and Reset**: Easily clear the input or reset the calculator to start a new calculation.

- **Responsive Design**: The calculator is fully responsive and works well on various screen sizes.

- **Immediate Results**: Get instant results as you input numbers and perform calculations.

## How to Use

1. Open the Simple Calculator in your web browser.

2. Use the on-screen buttons to input numbers and perform calculations.

3. Click the number buttons (0-9) to input numbers.

4. Click the operation buttons (+, -, *, /) to select the desired arithmetic operation.

5. Click the "=" button to calculate the result.

6. The result will be displayed in the calculator's display area.

7. To clear the input, click the "C" (Clear) button.

8. Enjoy using the Simple Calculator for quick arithmetic calculations!

## Technologies Used

- HTML: Structure of the calculator.
- CSS: Styling of the calculator.
- JavaScript: Logic for calculator functionality and UI interactions.
